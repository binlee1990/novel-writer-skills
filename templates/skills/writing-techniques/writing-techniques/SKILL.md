# Writing Techniques - 写作技巧教学

## Skill类型
- **类别**: writing-techniques
- **激活方式**: /write 命令自动激活
- **优先级**: 高（与 writing-balance 协同工作）

## 设计理念

**从正面教授写作技巧，而非负面禁止**

与 `writing-balance` 的关系：
- writing-balance: 评估"写得好不好"
- writing-techniques: 教授"怎么写更好"

## 功能概述

提供8个模块的写作技巧教学，帮助AI和用户掌握自然、生动、有层次的写作方法。

---

## 教学模块（8个）

### 1. 标点符号艺术

**核心原则**: 标点符号不仅是语法工具，更是节奏和情感的载体

#### 省略号的3种用法

**1.1 思考停顿**
```
他想了想......
→ ❌ AI化（机械停顿）

他想了想。
→ ✅ 自然（简洁有力）

他想了想，半天没说话。
→ ✅ 自然（用动作代替省略号）
```

**1.2 未尽之意**
```
"我...我不是故意的......"
→ ✅ 可用（表达犹豫）

"我不是故意的。"她低着头。
→ ✅ 更好（用动作烘托情绪）
```

**1.3 语音延长**
```
"啊——"
→ ✅ 用破折号而非省略号
```

**使用建议**:
- 每1000字最多3-5个省略号
- 优先用动作/表情代替
- 避免"想了想......"这种AI化用法

---

#### 破折号的节奏控制

**2.1 转折强调**
```
他打开门——愣住了。
→ ✅ 制造悬念感
```

**2.2 解释说明**
```
她笑了——那是一种很复杂的笑。
→ ⚠️ AI化（过度解释）

她笑了。笑得有点复杂。
→ ✅ 更自然（拆分句子）
```

**2.3 声音延长**
```
"不——"
→ ✅ 表达惊恐/拒绝
```

---

#### 句号vs逗号的节奏把控

**快节奏**（短句 + 句号）:
```
门开了。
他进来。
看了她一眼。
转身走了。

→ ✅ 紧张感、决绝感
```

**慢节奏**（长句 + 逗号）:
```
门缓缓打开，他走进来，看了她一眼，然后转身走了。

→ ⚠️ 缺乏停顿，一口气说完
```

**平衡节奏**:
```
门开了，他进来。
看了她一眼，转身走了。

→ ✅ 长短句结合，节奏自然
```

---

### 2. 句式变化技巧

**核心原则**: 避免连续相同句式，营造节奏变化

#### 2.1 长短句配合

**"长铺垫 + 短点题"模式**:
```
窗外的雨越下越大，雨点打在玻璃上发出噼啪声，
路上的行人打着伞匆匆走过，整条街笼罩在灰蒙蒙的雨雾中。
很冷。

→ ✅ 长句铺垫氛围，短句点题情绪
```

**"短起 + 长承 + 短转"模式**:
```
门开了。（短）
他站在门口，手里拿着一把伞，身上的衣服湿了大半，
头发上还滴着水。（长）
她愣住了。（短）

→ ✅ 节奏有起伏
```

---

#### 2.2 倒装强调

**正常语序**:
```
门开了。
→ 平铺直叙
```

**倒装语序**:
```
门，开了。
→ ✅ 强调"门"，制造悬念

开了，门。
→ ❌ 过度倒装，不自然
```

**适用场景**:
- 情绪转折点
- 悬念揭示前
- 强调某个关键动作/物品

---

#### 2.3 省略句的留白

**完整句**:
```
他走了，没有回头。
→ 正常表达
```

**省略句**:
```
他走了。
没回头。

→ ✅ 留白，更有力量感
```

**连续省略**:
```
门。
开了。
他。
进来。

→ ❌ 过度省略，变成电报体
```

**使用建议**:
- 每段最多1-2个省略句
- 用于情绪高潮或转折点
- 避免连续使用

---

### 3. 描写手法

**核心原则**: 五感交叉、动静结合、侧面烘托

#### 3.1 五感交叉描写

**只用视觉**（AI常见）:
```
她站在窗前，看着外面的雨。
月光照在她脸上，显得很美。

→ ⚠️ 单一感官，缺乏沉浸感
```

**五感交叉**:
```
她站在窗前。
雨水敲打玻璃，发出细碎的声音。（听觉）
空气里有股潮湿的味道。（嗅觉）
月光冷冷地打在她脸上。（视觉 + 触觉）

→ ✅ 多感官，沉浸感强
```

**五感分布建议**:
- 视觉: 40%（主导，但不唯一）
- 听觉: 20%（营造氛围）
- 触觉/嗅觉/味觉: 40%（细节，记忆点）

---

#### 3.2 动静结合

**只写静态**:
```
房间很安静。
她坐在椅子上。
桌上放着一杯咖啡。

→ ⚠️ 画面呆板
```

**动静结合**:
```
房间很安静。
她坐在椅子上，手指在桌面上轻轻敲着。（动）
咖啡还冒着热气。（动）

→ ✅ 画面有生命力
```

---

#### 3.3 侧面烘托

**直接描写**（AI常见）:
```
她很美。
→ ⚠️ 空洞无力
```

**侧面烘托**:
```
路过的人都在看她。
→ ✅ 用他人反应烘托

他第一次见到她，愣了好几秒。
→ ✅ 用他人反应烘托
```

**使用场景**:
- 描写人物魅力
- 描写场景震撼力
- 描写情绪强度

---

### 4. 对话真实化

**核心原则**: 让对话像真人说话，而非书面语

#### 4.1 语气词自然使用

**AI化对话**（书面语）:
```
"你怎么能这样？"她愤怒地说。
→ ⚠️ 生硬
```

**真人对话**:
```
"你怎么能这样啊？"
"你这样算什么嘛。"
"哎，你别这样。"

→ ✅ 语气词：啊、吧、呢、哦、嘛、哎
```

**使用建议**:
- 每3-5句对话中用1个语气词
- 不同角色用不同语气词（人物标签）
- 避免过度使用（每句话都有）

---

#### 4.2 未完成句

**完整句**:
```
"我不是故意的，我只是......"
→ ⚠️ AI化用法（省略号滥用）
```

**未完成句**:
```
"我不是...我只是......"
"我......"她说不下去了。

→ ✅ 表达犹豫、情绪激动
```

---

#### 4.3 口头禅/方言点缀

**标准对话**:
```
"我知道了。"
→ 中性，无个性
```

**有口头禅的对话**:
```
"得嘞，我知道了。"（北方）
"好的呀。"（南方）
"行，晓得了。"（西南）

→ ✅ 地域感、角色个性
```

**使用建议**:
- 主要角色可以有1-2个口头禅
- 不要所有角色都用同一个
- 适度使用，不要每句话都有

---

### 5. 节奏控制

**核心原则**: 不同场景用不同节奏

#### 5.1 快节奏（动作/冲突场景）

**特征**:
- 短句为主（<15字）
- 动词密集
- 少形容词
- 多用句号

**示例**:
```
门被踹开了。
他冲进来。
抓住她的手。
往外拖。

她挣扎。
尖叫。
没用。

→ ✅ 紧张感强
```

---

#### 5.2 慢节奏（描写/铺垫场景）

**特征**:
- 长句为主（>20字）
- 描写细腻
- 多感官
- 多用逗号

**示例**:
```
窗外的雨越下越大，雨点敲打着玻璃，
发出细碎的声音。她坐在窗边，手里端着一杯已经凉了的咖啡，
看着街上匆匆而过的行人，突然觉得有点累。

→ ✅ 营造氛围，沉浸感
```

---

#### 5.3 情绪递进（句子逐渐变短/变长）

**由长到短**（情绪爆发）:
```
她一直忍着，努力保持平静，告诉自己不要在意。
但是听到这句话，她再也忍不住了。
她抬起头。
看着他。
"你说什么？"

→ ✅ 句子越来越短，情绪逐渐爆发
```

**由短到长**（情绪平复）:
```
她哭了。
哭得很伤心。
哭了好一会儿，渐渐平静下来，擦了擦眼泪，深吸一口气，
然后抬起头，看着窗外的天空。

→ ✅ 句子逐渐变长，情绪逐渐平复
```

---

### 6. 留白艺术

**核心原则**: 不要说尽，留给读者想象空间

#### 6.1 段落间的空白

**无留白**:
```
他走了，她很难过，一个人坐在房间里哭。
第二天早上，她起床后发现他回来了。

→ ⚠️ 时间跳跃生硬
```

**有留白**:
```
他走了。
她一个人坐在房间里。

第二天早上。

门开了。
他回来了。

→ ✅ 空行制造时间跳跃感
```

---

#### 6.2 未说之意

**全部说出**（AI常见）:
```
他看了她一眼，眼神中充满了复杂的情绪，
有愤怒，有失望，也有一丝不舍。然后转身走了。

→ ⚠️ 过度解释
```

**留白**:
```
他看了她一眼。
转身走了。

→ ✅ 让读者自己体会眼神
```

---

#### 6.3 读者想象空间

**描写过细**:
```
她穿着一件白色的连衣裙，裙摆到膝盖，
腰间系着一条蓝色的腰带，脚上穿着一双黑色的高跟鞋。

→ ⚠️ 像产品说明书
```

**留白**:
```
她穿着白色连衣裙。
很好看。

→ ✅ 点到为止，让读者自己想象
```

---

### 7. 词汇选择原则

**核心原则**: 具体>抽象，动词优先，避免过度修饰

#### 7.1 具体>抽象

**抽象词**:
```
她喝了一杯饮料。
→ ⚠️ 模糊

她喝了一杯咖啡。
→ ✅ 更具体

她喝了一杯拿铁。
→ ✅ 最具体（生活细节）
```

**使用建议**:
- 优先用具体名词
- 品牌/型号可以增强真实感
- 但不要过度（变成广告）

---

#### 7.2 动词优先

**被动句**（AI常见）:
```
门被关上了。
→ ⚠️ 缺乏力量感
```

**主动句**:
```
她关上了门。
→ ✅ 更有力

门砰地关上了。
→ ✅ 最有力（动词 + 象声词）
```

---

#### 7.3 避免过度修饰

**过度修饰**:
```
非常特别极其好
→ ❌ 堆砌程度词

修长白皙的手指
→ ❌ 连续形容词
```

**适度修饰**:
```
很好
→ ✅ 简洁有力

白皙的手指
→ ✅ 一个形容词足够
```

---

### 8. AI痕迹识别与规避

**核心原则**: 控制频率，而非完全禁止

#### 8.1 AI高频词表（100个）

**情绪类**:
- 五味杂陈、百感交集、心如刀绞 → 替代：不是滋味、心里乱、心疼得要命

**动作类**:
- 缓缓、悠然、从容不迫 → 替代：慢慢、随意、不慌不忙

**程度类**:
- 极其、万分、异常 → 替代：很、特别

**转折类**:
- 然而、殊不知、岂料 → 替代：但是、没想到

**描写类**:
- 修长的、深邃的、璀璨的 → 替代：删除或简化

**使用建议**:
- 不完全禁用，但限制频率（≤2次/1000字）
- 优先用替代词
- 如果必须用，间隔至少500字

---

#### 8.2 AI典型句式（20种）

**1. "在...下"**
```
在月光的照耀下，她的脸庞显得格外动人。
→ ❌ AI化

月光打在她脸上。
她脸上有月光，挺好看的。
→ ✅ 改写

使用建议: ≤2次/1000字
```

**2. "仿佛...一般"**
```
他的速度快得仿佛一阵风一般。
→ ❌ AI化

他速度特别快。
他快得像阵风。
→ ✅ 简化比喻

使用建议: ≤2次/1000字
```

**3. "此时此刻"**
```
此时此刻，她的心情格外复杂。
→ ❌ AI化

她心情很复杂。
这会儿她心里乱得很。
→ ✅ 删除时间指示

使用建议: 完全避免
```

**4. "让...让..."**
```
温热的触感让她的思绪渐渐平静下来。
→ ❌ AI化（因果链）

杯子有点烫。她手暖了。
她心情慢慢平静了。
→ ✅ 拆分因果

使用建议: 拆分为独立句子
```

## 综合应用示例

### 场景：主角遇到危险（对比）

#### ❌ AI化写法

```
李明缓缓抬起头，深邃的眸子中闪过一丝凝重的神色。
他能够清晰地感受到，周围的气氛变得异常诡异，
仿佛有什么不好的事情即将发生一般。

然而，事情远比他想象的要复杂得多。殊不知，
在这片看似平静的树林深处，正有一双充满杀意的眼睛
在暗中观察着他的一举一动。此时此刻，危机四伏，
他却浑然不觉。
```

**问题**:
- 禁用词：缓缓、深邃、异常、然而、殊不知、此时此刻
- 禁用句式："仿佛...一般"、"在...下"
- AI化表达：清晰地感受到、一举一动、浑然不觉

---

#### ✅ 优化写法（应用8个模块）

```
李明抬头。（短句，节奏）

周围太安静了。（句式变化）

他突然意识到不对劲。猛地转身。（快节奏）

树林里，有什么在动。（留白，悬念）

糟了。（省略句，情绪爆发）
```

**应用的技巧**:
1. 标点符号：句号制造停顿感
2. 句式变化：短句为主，营造紧张
3. 节奏控制：快节奏，动词密集
4. 留白艺术：不明说危险，让读者自己感受
5. 词汇选择：具体动词（抬头、转身、动）

## 与 writing-balance 的协同

**writing-balance** 负责评估：
- 告诉你"句长分布不平衡"
- 告诉你"词汇重复度高"

**writing-techniques** 负责教学：
- 教你"怎么混合长短句"
- 教你"怎么用同义词"

**工作流程**:
1. 用户使用 `/write` 写作
2. AI应用 `writing-techniques` 的技巧创作
3. 写完后，`writing-balance` 自动评估
4. 如果评分低，给出改进建议
5. 用户参考 `writing-techniques` 的具体技巧调整

---

## 实施建议

1. **逐步学习**: 不要一次性应用所有技巧，先掌握1-2个模块
2. **实战练习**: 每写一章后，对照评估报告找问题
3. **形成习惯**: 逐渐内化这些技巧，不需要刻意思考
4. **保留个性**: 技巧是工具，不是枷锁，保留自己的风格
