# 卷级详细规划 Skill

**触发条件**：`/plan` 命令的 `$ARGUMENTS` 包含 `--detail vol-XX`（如 `--detail vol-01`）

**前置条件**：
1. 必须已有 `creative-plan.md`（全书规划），如没有则提示先执行 `/plan`
2. 从 `creative-plan.md` 中读取目标卷的概要信息（卷主题、起止章节、核心冲突）

## 卷级规划流程

### Step 1: 卷概要确认

从 `creative-plan.md` 中提取目标卷的信息，向用户确认：

```
📖 卷级详细规划：第 [X] 卷 — [卷名]
━━━━━━━━━━━━━━━━━━━━
📋 卷主题：[从 creative-plan.md 提取]
📊 章节范围：第 [A]-[B] 章（共 [N] 章）
⚡ 核心冲突：[从 creative-plan.md 提取]
🎯 卷目标：[本卷结束时要达到的状态]

确认以上信息正确？如需调整请说明。
```

### Step 2: 逐章规划

为本卷的每一章生成详细规划：

```markdown
### 第 [N] 章：[章节标题]

**章节定位**：[开篇/发展/转折/高潮/过渡/收束]
**字数预估**：[2000-4000 字]

**本章目标**：
1. [情节目标：推进什么情节线]
2. [角色目标：展示/发展什么角色]
3. [信息目标：释放什么信息给读者]

**场景规划**：
- 场景 1：[地点] — [事件概要]（约 [X] 字）
- 场景 2：[地点] — [事件概要]（约 [X] 字）

**在场角色**：[角色列表]

**爽点设计**：
- [爽点类型]：[具体描述]（强度：⭐⭐⭐）
- 如无爽点：标注「过渡章，无主要爽点」

**钩子设计**：
- 类型：[悬念/反转/信息/情感/时间压力/选择]
- 内容：[钩子具体内容]
- 强度：[⭐-⭐⭐⭐⭐⭐]
- 兑现计划：第 [M] 章

**伏笔操作**：
- 埋设：[新伏笔描述]（预计第 [M] 章回收）（伏笔链：[链名称]）
- 推进：[已有伏笔] → [本章推进方式]
- 回收：[已有伏笔] → [本章回收方式]（影响：major|moderate|minor）

**情绪节奏**：[高/中/低]
**与上一章的衔接**：[如何承接上一章结尾]
```

### Step 3: 卷级节奏总览

规划完所有章节后，输出节奏总览：

```
📊 第 [X] 卷节奏总览
━━━━━━━━━━━━━━━━━━━━

情绪曲线：
第 1 章 ████░░░░░░ 中
第 2 章 ██████░░░░ 中高
第 3 章 ████████░░ 高
第 4 章 ███░░░░░░░ 低（过渡）
第 5 章 █████░░░░░ 中
第 6 章 ██████████ 高潮
...

爽点分布：
第 1 章 ⭐⭐ 逆袭
第 3 章 ⭐⭐⭐ 打脸
第 5 章 ⭐⭐⭐⭐ 升级
第 6 章 ⭐⭐⭐⭐⭐ 大反转

钩子链：
第 1 章 → 悬念 ⭐⭐⭐ → 第 2 章兑现
第 2 章 → 信息 ⭐⭐ → 第 3 章兑现
第 3 章 → 反转 ⭐⭐⭐⭐ → 第 5 章兑现（跨章）
...

⚠️ 节奏警告：
- 第 4-5 章连续中低节奏，考虑在第 4 章末尾加强钩子
- 第 6 章高潮后建议第 7 章适当放缓
```

### Step 4: 写入 creative-plan.md

将卷级详细规划追加到 `creative-plan.md` 的对应卷章节中，或创建独立的 `vol-XX-detail.md` 文件。

**写入策略**：
- 如果 `creative-plan.md` 中已有该卷的详细规划 → 询问是否覆盖
- 如果 `creative-plan.md` 过长（> 500 行）→ 创建 `stories/*/spec/vol-XX-detail.md`
- 否则 → 追加到 `creative-plan.md` 的对应位置

### Step 5: 生成 tasks.md 条目

基于卷级规划，自动生成 `tasks.md` 中的写作任务：

```markdown
## 第 [X] 卷写作任务

- [ ] 第 [A] 章：[章节标题] — [一句话目标]
- [ ] 第 [A+1] 章：[章节标题] — [一句话目标]
...
- [ ] 第 [B] 章：[章节标题] — [一句话目标]
```

### Step 6: 灵感分配

在卷级详细规划完成后，自动将未使用的灵感分配到合适的章节。

**前置条件**：`notes/ideas.json` 存在且包含 `status=new` 的灵感

**分配流程**：

1. 读取所有 `status=new` 的灵感
2. 根据灵感的 `tags`、`category`、`relatedCharacters` 匹配章节计划
3. 生成灵感分配建议

**输出格式**：

```
💡 灵感分配建议
━━━━━━━━━━━━━━━━━━━━

未使用灵感：[N] 条

建议分配：
第 26 章 ← 💡 [灵感A]（标签匹配：#战斗）
第 28 章 ← 💡 [灵感B]（角色匹配：师姐）
第 30 章 ← 💡 [灵感C]（类别匹配：转折）

未匹配灵感（[M] 条）：
- 💡 [灵感D]（标签：#结局）→ 可能适合后续卷
- 💡 [灵感E]（标签：#番外）→ 可能适合番外篇

是否确认分配？[Y/N]
```

**确认后操作**：
- 更新对应灵感的 `status` 为 `planned`
- 更新对应灵感的 `relatedChapters` 添加分配的章节号
- 写回 `notes/ideas.json`

## 多卷批量规划（`--detail vol-XX-YY`）

**触发条件**：`$ARGUMENTS` 包含 `--detail vol-XX-YY`（如 `--detail vol-01-03`）

**前置条件**：
1. 必须已有 `creative-plan.md`（全书规划）
2. 从 `creative-plan.md` 中读取目标范围内所有卷的概要信息

### Step 1: 范围确认

```
📖 多卷批量规划：第 [X] - [Y] 卷
━━━━━━━━━━━━━━━━━━━━
📋 规划范围：
| 卷 | 卷名 | 章节范围 | 核心冲突 |
|----|------|---------|---------|
| 第 X 卷 | [卷名] | 第 A-B 章 | [冲突] |
| 第 X+1 卷 | [卷名] | 第 C-D 章 | [冲突] |
| ... | ... | ... | ... |
| 第 Y 卷 | [卷名] | 第 E-F 章 | [冲突] |

确认以上信息正确？如需调整请说明。
```

### Step 2: 逐卷规划

对范围内的每一卷，执行上述 Step 2-4 流程（逐章规划 + 节奏总览 + 写入）。

**卷间衔接设计**：
在规划每一卷时，额外关注：
- 上一卷结尾钩子与本卷开头的衔接
- 跨卷伏笔的延续和推进
- 角色状态的跨卷连续性
- 情绪曲线的跨卷递进

### Step 3: 跨卷节奏对比

所有卷规划完成后，输出跨卷对比总览：

```
📊 跨卷节奏对比
━━━━━━━━━━━━━━━━━━━━

| 维度 | 第 X 卷 | 第 X+1 卷 | ... | 第 Y 卷 |
|------|---------|-----------|-----|---------|
| 章节数 | [N] | [N] | ... | [N] |
| 爽点数 | [N] | [N] | ... | [N] |
| 最高情绪 | [描述] | [描述] | ... | [描述] |
| 卷末钩子 | [类型] ⭐⭐⭐ | [类型] ⭐⭐⭐⭐ | ... | [类型] ⭐⭐⭐⭐⭐ |

⚠️ 跨卷警告：
- [如有节奏递进问题]
- [如有爽点密度不均]
- [如有卷间衔接问题]
```

### Step 4: 生成批量 tasks.md

基于所有卷的规划，批量生成 `tasks.md` 中的写作任务。
