# 科幻小说创作规范

## 快速参考

| 元素 | 指导原则 | 位置 |
|------|---------|------|
| **日常世界** | 展示科技背景下的日常生活 | 前 10% |
| **触发事件** | 科技异变/发现/危机 | 10-15% |
| **探索与危机** | 理解新规则，问题升级 | 15-50% |
| **中点反转** | 关键发现改变理解 | 50% |
| **代价与抉择** | 技术代价显现，人性选择 | 50-85% |
| **高潮** | 解决核心冲突 | 85-95% |
| **新常态** | 世界因此改变 | 最后 5% |

## 子类型识别

| 子类型 | 关键词 | 核心特征 | 知识库重点 |
|--------|--------|---------|-----------|
| 硬科幻 | 物理、方程、技术细节 | 科学准确性 | 技术规则严谨性 |
| 软科幻 | 社会、文明、人性 | 人文关怀 | 社会推演合理性 |
| 赛博朋克 | 黑客、义体、企业 | 高科技低生活 | 反差与阶级冲突 |
| 太空歌剧 | 帝国、舰队、星系 | 宏大叙事 | 史诗感与政治博弈 |
| 末日科幻 | 废墟、生存、重建 | 文明崩溃 | 生存逻辑与人性考验 |
| 近未来 | AI、基因编辑、VR | 当前技术延伸 | 技术推演可信度 |

## 核心惯例

1. **科学基底**：核心科技必须有规则和限制，不能万能
2. **世界观自洽**：技术规则一旦确立，全文不可违反
3. **人文内核**：科技是背景，人性选择是主题
4. **信息投放**：世界观通过故事展示，避免大段说明
5. **技术代价**：任何技术进步都有副作用或社会代价
6. **陌生化效果**：用未来视角重新审视熟悉事物

## 常见陷阱

| 陷阱 | 表现 | 修正方向 |
|------|------|---------|
| 技术万能 | 科技没有限制和代价 | 明确规则边界和副作用 |
| 规则违反 | 为剧情方便打破已建立的设定 | 在规则内寻找创意解法 |
| 信息倾倒 | 开头大段世界观说明 | 通过角色行动自然展示 |
| 工具人角色 | 角色只是操作机器的载体 | 回归人性选择和情感 |
| 现代思维 | 未来人用现代人的方式思考 | 思维方式应匹配技术环境 |
