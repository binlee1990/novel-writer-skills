# 惊悚小说创作规范

## 快速参考

| 元素 | 指导原则 | 位置 |
|------|---------|------|
| **日常与不安** | 看似正常但有微妙异常 | 前 10% |
| **触发事件** | 威胁首次显现 | 10-15% |
| **第一次打击** | 威胁变得真实 | 25% |
| **中点反转** | 真正的威胁远超想象 | 50% |
| **最黑暗时刻** | 主角几乎被击败 | 75% |
| **高潮对决** | 最终对抗 | 90-95% |
| **余波** | 代价与新常态 | 最后 5% |

## 激活条件

以下关键词触发本 Skill：
- 惊悚、thriller、紧张、恐惧
- 追杀、逃亡、绑架、暗杀
- 阴谋、conspiracy、悬疑（偏惊悚方向）
- 倒计时、生存、威胁、危机

## 与悬疑（mystery）的区分

| 维度 | 悬疑 (Mystery) | 惊悚 (Thriller) |
|------|---------------|-----------------|
| 核心驱动 | 好奇心（谁干的？） | 恐惧（能活下来吗？） |
| 信息方向 | 向后看（还原真相） | 向前看（阻止威胁） |
| 主角角色 | 调查者 | 目标/猎物 |
| 节奏 | 层层剥茧 | 不断升级 |
| 读者体验 | 智力满足 | 肾上腺素 |

**混合类型**：悬疑惊悚（Mystery Thriller）同时包含"谁干的"和"能否活下来"两条线。

## 子类型识别

| 子类型 | 关键词 | 核心特征 |
|--------|--------|---------|
| 心理惊悚 | 精神、操控、不可靠叙述 | 威胁来自人心 |
| 政治惊悚 | 政府、阴谋、权力 | 涉及权力机构 |
| 法律惊悚 | 法庭、律师、正义 | 法律体系内的博弈 |
| 科技惊悚 | 黑客、AI、病毒 | 技术失控 |
| 生存惊悚 | 荒野、灾难、孤立 | 极端环境求生 |

## 核心惯例

1. **滴答钟**：必须有时间压力，倒计时是最强张力工具
2. **风险升级**：每章风险都比上一章更高，永不降级
3. **信息不对称**：让读者知道主角不知道的危险
4. **喘息节奏**：高压后安排短暂安全，但埋下新的不安
5. **可信威胁**：反派必须在早期证明自己的危险性
6. **主角脆弱**：主角不能是超人，脆弱性是惊悚感来源

## 常见陷阱

| 陷阱 | 表现 | 修正方向 |
|------|------|---------|
| 狼来了 | 威胁从不兑现 | 让威胁真正伤害到角色 |
| 主角无敌 | 主角太强没有危险感 | 给主角明确的弱点和限制 |
| 巧合救场 | 靠运气脱险 | 主角靠智慧和代价脱险 |
| 持续高压 | 连续高压导致麻木 | 安排喘息但埋下不安种子 |
| 虎头蛇尾 | 高潮不如中段精彩 | 确保高潮是全书张力最高点 |

## 推荐节奏

```
张力曲线（理想）：

★★ ─── 日常
  ★★★ ── 触发
    ★★★★ ── 第一次打击
      ★★★ ── 喘息
        ★★★★ ── 升级
          ★★★★★ ── 中点反转
            ★★★★ ── 喘息
              ★★★★★ ── 最黑暗时刻
                ★★★★★ ── 高潮
                  ★★★ ── 余波
```

**关键**：整体趋势向上，喘息不低于前一个低点。
