# Show, Don't Tell 写作知识库

## 快速参考（Quick Reference）

"Show, Don't Tell"（展示，别讲述）是小说写作的黄金法则——不要直接告诉读者角色的感受或情境，而是通过具体细节让读者自己推导。

**核心理念**：
```
Tell（讲述）= 直接说出结论
Show（展示）= 展现细节，让读者得出结论
```

**基本公式**：
```
Tell: "她很生气"
Show: "她一拳砸在桌上，杯子跳了起来"
```

**Show 的三个层次**：

| 层次 | 方法 | 示例 |
|------|------|------|
| 1级 | 行动展示 | 她摔门离开 |
| 2级 | 感官细节 | 她的手在颤抖 |
| 3级 | 环境反应 | 房间里的人都僵住了 |

**何时 Tell**：
- 过渡性信息（"三天后……"）
- 次要情节（"他度过了平淡的一周"）
- 节奏需要（避免所有细节都展示）

**不要过度 Show**：
- ❌ 每个细节都展示（拖沓）
- ❌ 简单信息也展示（浪费篇幅）

## 核心原则（Core Principles）

### 1. Show vs Tell 的本质区别

#### Tell：直接说出结论

**特征**：
- 使用抽象词汇
- 概括性描述
- 作者直接告诉读者

**常见 Tell 词汇**：
- 情感词：生气、悲伤、快乐、害怕、紧张
- 状态词：疲惫、兴奋、困惑、尴尬
- 性格词：勇敢、懦弱、聪明、愚蠢

❌ **Tell 示例**：
```
她很伤心。
他很生气。
房间很乱。
她是一个善良的人。
```

#### Show：展现细节，让读者推导

**特征**：
- 使用具体细节
- 描写可见/可感的内容
- 读者自己得出结论

✅ **Show 示例**：
```
【Tell】她很伤心。
【Show】她坐在角落，膝盖抱在胸前，脸埋在臂弯里。

【Tell】他很生气。
【Show】他的拳头握得指节发白，太阳穴上青筋暴起。

【Tell】房间很乱。
【Show】衣服堆在椅子上，桌上的咖啡杯里长了霉，地板上散落着用过的纸巾。

【Tell】她是一个善良的人。
【Show】她脱下自己的外套，披在流浪汉身上。
```

**为什么 Show 更好**：
- **读者参与**：让读者推理，产生互动
- **更生动**：具体细节比抽象词汇更有画面感
- **更可信**：读者自己得出的结论更有说服力
- **多层含义**：同一个细节可以传达多重信息

---

### 2. Show 的三个工具

#### 工具 1：行动展示（Action Shows）

**原则**：通过角色做什么来展示他们是谁、感受什么。

❌ **Tell**：
```
他很紧张。
```

✅ **Show（行动）**：
```
他来回踱步，不停地看手表。
```

**更多示例**：

| Tell | Show（行动） |
|------|-------------|
| 她不相信他 | 她后退一步，双手环抱胸前 |
| 他很自信 | 他大步走进房间，占据了中心位置 |
| 她很疲惫 | 她坐下时重重地叹了口气，鞋子踢掉了也懒得管 |
| 他很紧张 | 他的笔在纸上画着无意义的圈 |

**行动的多层含义**：

一个行动可以同时展示多种信息。

```
【行动】她握紧了手里的照片，指甲嵌进掌心。

【展示的信息】
- 情感：愤怒/痛苦
- 对照片的执念
- 身体张力（指甲嵌进）
- 克制（没有撕掉，而是握紧）
```

#### 工具 2：身体语言（Body Language Shows）

**原则**：通过身体的微小变化展示内在情感。

**情感与身体语言对照表**：

| 情感 | Tell | Show（身体语言） |
|------|------|----------------|
| 害怕 | 她很害怕 | 她的瞳孔放大，呼吸变浅 |
| 撒谎 | 他在撒谎 | 他的目光飘向左上方，手摸了摸鼻子 |
| 吸引 | 她被他吸引 | 她不自觉地拨了拨头发，身体微微前倾 |
| 厌恶 | 他很厌恶 | 他的嘴角下压，鼻子皱了一下 |
| 悲伤 | 她很悲伤 | 她的肩膀垮下来，眼眶发红 |

**微表情的力量**：

```
【Tell】他假装不在乎，但其实很受伤。

【Show】
"无所谓。"他耸耸肩。
但他的手攥紧了，指节发白。
```

**身体语言组合**：

真实的情感通常不是单一信号。

```
【紧张的完整展示】
他的腿在桌下不停抖动。
手心出汗，他在裤子上蹭了蹭。
目光不敢直视对方，看向窗外。
喉结滚动了一下。
```

#### 工具 3：感官细节（Sensory Details Show）

**原则**：通过五感（视觉、听觉、嗅觉、味觉、触觉）让读者"进入"场景。

**五感运用**：

**视觉（最常用）**：
```
【Tell】房间很旧。
【Show】墙纸在角落剥落，露出发黄的墙面。
```

**听觉**：
```
【Tell】夜晚很安静。
【Show】只有钟表的滴答声和她自己的呼吸。
```

**嗅觉**：
```
【Tell】这里是医院。
【Show】空气中弥漫着消毒水的味道。
```

**味觉**：
```
【Tell】她很紧张。
【Show】她咬了咬嘴唇，尝到了血的铁锈味。
```

**触觉**：
```
【Tell】天气很冷。
【Show】冷风钻进衣领，她打了个哆嗦。
```

**感官组合**：

最有力的 Show 组合多种感官。

```
【场景：废弃工厂】

【Tell】这里很可怕。

【Show - 多感官】
锈迹斑斑的铁门发出刺耳的嘎吱声（听觉）。
空气中是潮湿和霉菌的气味（嗅觉）。
她的手碰到墙壁，冰冷粗糙（触觉）。
微弱的光从破窗照进来，照亮地上的碎玻璃（视觉）。
```

### 3. Show 的层次理论

#### 层次 1：基础 Show - 替换抽象词

最简单的 Show：用具体细节替换抽象词汇。

| 抽象（Tell） | 具体（Show） |
|-------------|-------------|
| 很热 | 汗水顺着后背流下 |
| 很穷 | 鞋底破了个洞 |
| 很老 | 满脸皱纹，手上老年斑 |
| 很乱 | 地上踩满了烟蒂 |

#### 层次 2：进阶 Show - 间接展示

不直接描述事物，而是通过其他事物反映。

```
【目标】展示房间很冷

【基础 Show】房间里结了冰。
【进阶 Show】她呼出的气在空中凝成白雾。
→ 通过呼吸间接展示寒冷
```

**更多示例**：

| 要展示的 | 间接方式 |
|---------|---------|
| 时间流逝 | 烟灰缸里堆满了烟蒂 |
| 他很有钱 | 服务员对他毕恭毕敬 |
| 她很美 | 房间里所有男人的目光都转向她 |
| 危险临近 | 鸟儿突然停止了鸣叫 |

#### 层次 3：高级 Show - 多重含义

一个细节展示多层信息。

```
【细节】她打开冰箱，里面只有一瓶过期的牛奶和半个发霉的苹果。

【展示的信息】
1. 经济困难（买不起食物）
2. 生活混乱（食物过期）
3. 孤独（没有人关心她）
4. 可能的抑郁（失去生活动力）
```

### 4. 何时该 Tell

**重要**：Show 不是绝对的，有时 Tell 更好。

#### 情况 1：过渡性信息

不重要的时间/地点转换。

✅ **适合 Tell**：
```
三天后，她到达了北京。
```

❌ **过度 Show**（浪费篇幅）：
```
第一天，她坐上火车……（500 字）
第二天，火车穿过平原……（500 字）
第三天，终于看到北京的城市轮廓……（500 字）
→ 1500 字，没有实质内容
```

#### 情况 2：次要信息

对主情节不重要的背景。

✅ **适合 Tell**：
```
他度过了平淡的一周。
```

❌ **过度 Show**：
```
周一，他起床，吃早餐……（每天都详细描写）
→ 浪费篇幅在次要内容上
```

#### 情况 3：节奏需要

有时需要快速推进，Tell 更快。

```
【场景】需要快速建立背景

✅ Tell: 她在这个城市已经生活了十年。

❌ Show: 她回忆起十年前第一次来到这个城市……（展开 2000 字）
→ 如果背景不重要，用 Tell 更有效率
```

#### 情况 4：对比效果

在一片 Show 中，偶尔 Tell 可以制造对比。

```
【大部分 Show 后】
她的手在颤抖，额头冒汗，眼神闪烁不定。
她撒谎了。← Tell，但在大量 Show 后，这个 Tell 有力
```

## 实践应用（Practical Application）

### 在 /specify 阶段

**定义 Show/Tell 比例**

```markdown
## Show/Tell 原则

### 整体比例
- Show: 70-80%（重要场景、情感时刻）
- Tell: 20-30%（过渡、次要信息）

### 必须 Show 的内容
- 角色情感（生气、悲伤等）
- 重要场景（初遇、告别、冲突）
- 角色性格展示
- 环境氛围营造

### 可以 Tell 的内容
- 时间过渡（"三天后"）
- 次要角色的背景
- 不重要的日常活动
```

### 在 /plan 阶段

**标记需要重点 Show 的场景**

```markdown
第 5 章：初遇

【重点 Show】
- 林晓第一次见到张伟
- 通过细节展示：
  - 林晓的紧张（身体语言）
  - 张伟的气场（环境反应）
  - 双方的第一印象（行动和对话）

【可以 Tell】
- 林晓如何到达会议室（"她提前十分钟到了"）
- 会议室的常规设施
```

### 在 /write 阶段

**写作时自查**

**步骤 1：找出所有 Tell**

搜索以下词汇：
- 很（很生气、很高兴）
- 感到（感到悲伤、感到害怕）
- 是一个……的人（是一个善良的人）
- 看起来（看起来很紧张）

**步骤 2：评估是否需要改为 Show**

```
【发现 Tell】她很生气。

【评估】
- 这是重要情感吗？是
- 有篇幅展示吗？有
- 读者需要感受吗？是
→ 结论：应该改为 Show

【改写】
她一言不发地走到窗边，双臂环抱胸前。
下巴绷紧，目光死死盯着窗外。
```

**步骤 3：检查是否过度 Show**

```
【发现过度 Show】
他起床，洗脸，刷牙（每一步都详细描写 300 字）……

【评估】
- 这对情节重要吗？否
- 读者需要知道这些细节吗？否
→ 结论：可以 Tell 或完全省略

【改写】
他匆匆洗漱后出门。
```

### 在 /analyze 阶段

**分析 Show/Tell 比例**

```markdown
## Show/Tell 分析

### 统计
- 纯 Tell 句子：50 句
- Show 段落：120 段
- 比例：Show 70% vs Tell 30% ✓

### 问题段落

【第 3 章第 2 段】
"她很伤心，很失望。"（Tell）
- 建议：这是重要情感场景，应该 Show
- 改写：用身体语言和行动展示

【第 10 章第 5 段】
"她慢慢走到门前，手放在门把上，深呼吸，转动门把……"（过度 Show）
- 建议：这是次要过渡，可以简化
- 改写："她推开门。"

### 优秀段落

【第 7 章第 3 段】
"他的手机在桌上震动，但他没有看。又震动一次，还是没看。第三次时，他拿起来，看了一眼，然后关机。"
- 分析：通过行动展示内心挣扎，优秀 Show ✓
```

## 常见陷阱（Common Pitfalls）

### 1. 过度依赖情感词汇

**问题**：直接使用"生气"、"悲伤"、"害怕"等词。

❌ **糟糕示例**：
```
她很生气，非常生气，气得不行。
```

✅ **改进示例**：
```
她的拳头攥得指节发白。
一言不发地盯着他。
太阳穴上青筋暴起。
```

**常见情感词及 Show 替代**：

| Tell | Show 示例 |
|------|----------|
| 生气 | 拳头紧握/砰地关门/瞪视 |
| 悲伤 | 眼眶发红/声音哽咽/肩膀颤抖 |
| 害怕 | 后退一步/瞳孔放大/手心出汗 |
| 高兴 | 嘴角上扬/眼睛发亮/步伐轻快 |
| 紧张 | 来回踱步/咬嘴唇/手指敲桌 |

### 2. 告诉读者角色的性格

**问题**：直接说"他是一个……的人"。

❌ **糟糕示例**：
```
林晓是一个善良、勇敢、聪明的人。
```

✅ **改进示例**：
```
【善良】她把午餐分了一半给流浪汉。
【勇敢】"我去。"她第一个站出来。
【聪明】"等等，这里有蹊跷。"她指着看似无关的两个细节。
→ 通过行动展示性格
```

**性格展示原则**：
```
不要说角色是什么样的人
展示角色做了什么样的事
让读者自己判断
```

### 3. 过度 Show 次要内容

**问题**：对不重要的细节也详细展示。

❌ **糟糕示例**：
```
她打开衣柜，里面挂着十几件衣服。她的手指从第一件滑到第二件，
然后是第三件。她在心里权衡，这件太正式了，那件又太随意……
（500 字详细描写选衣服过程）
最终，她选了一件蓝色连衣裙。
```

✅ **改进示例**：
```
她换上蓝色连衣裙，出门了。
→ 除非选衣服本身很重要（如第一次约会），否则 Tell 即可
```

**判断标准**：
```
重要 → Show
不重要 → Tell 或省略
```

### 4. "看起来" / "似乎" 的滥用

**问题**：用"看起来"、"似乎"等词，实际还是在 Tell。

❌ **假 Show**：
```
他看起来很生气。
她似乎很悲伤。
```

**分析**：
- 虽然用了"看起来"，但还是在告诉读者结论
- "很生气"、"很悲伤"仍是抽象词汇

✅ **真 Show**：
```
他的脸涨红了，拳头攥紧。
她的眼眶红了，声音颤抖。
```

### 5. 解释角色行为的动机

**问题**：做完行为后，再解释为什么这样做。

❌ **糟糕示例**：
```
她转身离开，因为她很生气。
他拒绝了，因为他不信任对方。
```

**分析**：
- "因为"后面的内容是 Tell
- 应该让行为本身展示动机

✅ **改进示例**：
```
她转身离开，砰地一声摔上门。
→ 行为本身展示了愤怒，不需要说"因为生气"

他后退一步，双手环抱胸前："我拒绝。"
→ 身体语言展示了不信任
```

### 6. 过度描写每个细节

**问题**：事无巨细，读者疲劳。

❌ **过度 Show**：
```
她伸出右手，五指张开，然后慢慢握住门把。
门把是金属的，有些冰冷。她能感觉到门把的纹路。
她开始转动门把，先是顺时针转了 30 度，
然后感觉到里面的锁芯在移动……
（200 字描写开门）
```

✅ **适度 Show**：
```
她推开门。
→ 除非开门本身很重要（如紧张时刻），否则无需详细描写
```

**平衡原则**：
```
重要动作/情感 → 详细 Show
日常动作 → 简洁 Show 或 Tell
```

## 示例分析（Examples）

### 案例 1：情感的 Show vs Tell

**场景**：角色得知亲人去世

❌ **全 Tell 版本**：
```
林晓接到电话，得知父亲去世了。
她非常悲伤，感到难以置信。
她哭了很久，觉得天都塌了。
```

**分析**：
- 全是抽象词汇（悲伤、难以置信、哭）
- 读者无法感同身受
- 缺乏画面感

✅ **Show 版本**：
```
手机从林晓手中滑落，砰地一声摔在地上。

她盯着破碎的屏幕，嘴唇张开，但没有声音。

膝盖发软。

她慢慢坐到地上，背靠着墙。

许久，她的肩膀开始颤抖。

无声地颤抖。
```

**分析**：
- 通过身体反应展示（手机滑落、膝盖发软）
- 通过行动展示（坐到地上）
- 通过细节展示（无声地颤抖）
- 读者自己感受到悲伤的重量

### 案例 2：性格的 Show vs Tell

**目标**：展示角色"善良"

❌ **Tell 版本**：
```
林晓是一个善良的人，总是帮助别人。
大家都说她心地好。
```

✅ **Show 版本**：
```
【场景 1】
流浪汉坐在地铁口，破碗里只有几枚硬币。
林晓停下来，从包里拿出午餐盒："我吃过了，给你吧。"

【场景 2】
新来的实习生打翻了咖啡，溅到客户的文件上。
所有人都屏住呼吸。
林晓站起来："是我碰的，我来重新打印。"

【场景 3】
她看到老人在公交车上站着，周围的人都低头玩手机。
她站起来："您坐。"
```

**分析**：
- 通过三个具体行动展示善良
- 比直接说"善良"更有说服力
- 读者自己得出结论："她是个好人"

### 案例 3：环境的 Show vs Tell

**场景**：废弃工厂

❌ **Tell 版本**：
```
这是一个废弃的工厂，很破旧，很可怕。
到处都是垃圾，空气很差。
```

✅ **Show 版本（多感官）**：
```
【视觉】
铁门锈迹斑斑，一推就掉下来一片锈皮。
地上的玻璃碎片在微弱的光下闪烁。

【听觉】
她的脚步声在空荡的车间里回荡。
远处传来不知名的滴水声。

【嗅觉】
空气中是霉菌和机油的混合气味。

【触觉】
墙面湿冷粘腻，她赶紧收回手。
```

**分析**：
- 不说"破旧"、"可怕"
- 通过具体细节让读者感受
- 多感官组合，沉浸感强

### 案例 4：适度 Tell 的例子

**场景**：时间过渡

✅ **适度 Tell**：
```
三个月过去了。

林晓已经完全适应了新工作。← Tell，但合理

这天早上，她像往常一样走进办公室——
却发现气氛不对。← Show，重要时刻
```

**分析**：
- 过渡用 Tell（三个月、适应工作）
- 重要时刻用 Show（气氛不对）
- 平衡效率和生动性

## 进阶技巧（Advanced Techniques）

### 1. 间接 Show（Indirect Showing）

**技巧**：不直接描述事物，通过周围事物反映。

**示例 1：展示角色很有钱**

❌ **直接 Show**：
```
他穿着名牌西装，戴着劳力士手表。
```

✅ **间接 Show**：
```
服务员看到他，立刻毕恭毕敬地迎上来。
"先生，您的老位置已经准备好了。"
```

**示例 2：展示角色很美**

❌ **直接 Show**：
```
她有大眼睛，高鼻梁，完美的身材。
```

✅ **间接 Show**：
```
她走进房间，谈话声停顿了一秒。
所有人的目光都转向她，然后又若无其事地移开。
```

**为什么间接更好**：
- 更微妙
- 展示他人反应 = 更可信
- 避免直接描述的尴尬

### 2. 对比 Show（Contrast Showing）

**技巧**：通过前后对比展示变化。

**示例：展示角色的改变**

```
【故事开始】
她总是低着头走路，碰到人就道歉。

【故事结束】
她直视对方的眼睛："不，我不同意。"
→ 通过对比展示成长，无需说"她变勇敢了"
```

### 3. 细节堆叠（Detail Stacking）

**技巧**：用多个小细节组合，展示一个大概念。

**示例：展示贫穷**

```
不要只说："他很穷。"

细节堆叠：
- 鞋底破了个洞
- 外套的袖口磨得发白
- 他只点了最便宜的菜
- 钱包里只有几枚硬币
- 手机是五年前的旧款
→ 多个细节组合 = 强烈印象
```

### 4. 环境作为情感镜子（Pathetic Fallacy）

**技巧**：用环境反映角色情感。

```
【角色悲伤】
外面开始下雨。
雨水顺着窗户流下，像眼泪。

【角色孤独】
房间里只有钟表的滴答声。
窗外的街道空无一人。

【角色希望】
第一缕阳光照进房间。
鸟儿开始鸣叫。
```

**警告**：不要过度使用，否则会显得刻意。

### 5. 留白的艺术（Showing by Not Showing）

**技巧**：有时不说出来，反而更有力。

**示例**：

```
【要展示】他们做了X事（但X很敏感）

【不直接展示】
切换到第二天早上。
她坐在床边，默默整理衣服。
他在厨房做早餐，谁也没说话。
→ 读者自己推测发生了什么
```

## 检查清单（Checklist）

### Show/Tell 检查

#### 识别 Tell
- [ ] 搜索"很"（很生气、很累）
- [ ] 搜索"感到"（感到悲伤）
- [ ] 搜索"是一个……的人"
- [ ] 搜索情感词汇（生气、悲伤、害怕）

#### 评估是否改为 Show
- [ ] 这是重要情感/场景吗？
- [ ] 有篇幅展示吗？
- [ ] 读者需要感受吗？
- [ ] 如果3个都是"是"→ 改为 Show

#### Show 质量检查
- [ ] 是否用了具体细节（而非抽象词）？
- [ ] 是否使用了身体语言/行动？
- [ ] 是否调动了感官（视觉、听觉等）？
- [ ] 读者能自己推导出结论吗？

#### 避免过度 Show
- [ ] 是否 Show 了不重要的内容？
- [ ] 是否每个细节都展示（拖沓）？
- [ ] 次要内容可以 Tell 或删除吗？

### 场景层面检查

#### 重要场景（必须 Show）
- [ ] 角色初遇
- [ ] 重要冲突
- [ ] 情感爆发
- [ ] 关键选择
- [ ] 高潮时刻

#### 可以 Tell 的内容
- [ ] 时间过渡
- [ ] 次要角色背景
- [ ] 不重要的日常活动
- [ ] 快速推进情节

## 总结（Summary）

**Show, Don't Tell 的黄金法则**：

1. **Show 重要的，Tell 次要的**
2. **用具体细节替代抽象词汇**
3. **展示行动和身体语言，而非情感词汇**
4. **调动五感，让读者"进入"场景**
5. **让读者自己推导结论**

**Show 的三个工具**：

| 工具 | 方法 | 示例 |
|------|------|------|
| 行动 | 角色做什么 | 她摔门离开 |
| 身体语言 | 微表情、姿态 | 拳头紧握、瞳孔放大 |
| 感官细节 | 五感描写 | 消毒水的气味、冰冷的触感 |

**何时 Tell**：
- 过渡信息（"三天后"）
- 次要内容（不重要的背景）
- 节奏需要（快速推进）

**最重要的**：
```
Show ≠ 事无巨细描写每个细节
Show = 用具体细节展示重要内容
让读者参与推理
```

**最终测试**：
删除所有情感词汇（生气、悲伤、害怕）。
读者仍能感受到角色的情感吗？
如果能，你的 Show 成功了。

**记住**：
Show 让读者**体验**故事
Tell 让读者**了解**故事
体验比了解更有力量。
