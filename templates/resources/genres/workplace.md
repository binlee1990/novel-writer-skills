# 职场小说创作知识库

## 快速参考（Quick Reference）

职场小说的核心是**行业真实与人际博弈**。

**定义性特征**：
1. **行业细节**：真实可信的职场环境和专业知识
2. **职场政治**：权力博弈、站队选择、利益交换
3. **个人成长**：从职场新人到行业精英的蜕变

**关键公式**：
```
职场小说 = 行业真实感 + 职场博弈 + 专业能力 + 人性考验
```

**类型划分**：
- **商战**：企业竞争、并购重组、资本运作
- **官场**：体制内晋升、政治智慧、为民情怀
- **金融**：投行、基金、股市、风险投资
- **法律**：律所、法庭、案件博弈
- **医疗**：医院、医患关系、医术与医德
- **互联网**：创业、产品、技术、融资

## 核心原则（Core Principles）

### 1. 行业真实性原则

**核心**：职场小说的读者很多是业内人士，细节失真会直接劝退。

**如何实现**：
- 了解行业基本术语和工作流程
- 薪资水平、职级体系要符合现实
- 工作内容描写要具体（不要只写"他加班到很晚"）
- 行业潜规则要有分寸（揭露但不猎奇）

❌ **错误示例**：刚毕业的实习生直接参与公司最核心的并购案，独立完成尽职调查。
✅ **正确示例**：作为实习生，她的工作是整理数据和打印文件。但她在复印材料时注意到了一个所有人都忽略的数字异常。

---

### 2. 职场博弈原则

**核心**：职场不是非黑即白，灰色地带才是最真实的。

**博弈层次**：
- **明面博弈**：业绩竞争、方案PK、竞标
- **暗线博弈**：站队、信息差、资源争夺
- **自我博弈**：原则与利益的冲突、理想与现实的妥协

**权力结构**：
- 直属上司（最直接的影响者）
- 跨部门关系（协作与竞争并存）
- 高层博弈（派系、路线之争）
- 外部势力（客户、竞对、监管）

**职场潜规则**：
- 会议上的发言顺序暗藏玄机
- 邮件抄送名单就是权力地图
- "我再考虑考虑"往往意味着拒绝
- 被边缘化的信号：不再被邀请参加核心会议

---

### 3. 专业能力原则

**核心**：主角的成功必须建立在专业能力之上，不能只靠人际关系。

**能力展现方式**：
- 用专业知识解决具体问题（而非笼统的"他很厉害"）
- 展示思维过程（分析、判断、决策的逻辑链）
- 专业能力有边界（不是什么都会）
- 学习能力比现有能力更重要

**技巧**：
- 用外行能理解的方式解释专业内容
- 通过对话自然带出专业知识
- 用结果验证能力（方案被采纳、项目成功）

---

### 4. 现实主义原则

**核心**：职场小说要反映真实的职场生态，不美化也不丑化。

**真实的职场**：
- 加班是常态，但不是所有加班都有意义
- 升职不只看能力，还看时机、关系、运气
- 同事关系复杂，既有竞争也有真诚的帮助
- 离职跳槽是正常选择，不是背叛

**避免极端**：
- 不要把职场写成宫斗（过度阴暗）
- 不要把职场写成童话（过度美好）
- 真实的职场是灰色的，有温情也有残酷

## 实践应用（Practical Application）

### 职场场景工具箱

**经典场景**：
- 面试：第一印象、压力面试、群面博弈
- 会议室：方案汇报、意见分歧、领导拍板
- 茶水间：八卦、非正式信息交换、试探
- 团建/饭局：酒桌文化、非正式社交、真心话
- 述职/考核：年终总结、KPI博弈、晋升答辩

### 职场对话特点

```
✅ 好的职场对话：
"这个方案，你们部门内部讨论过了吗？"
"讨论过了，王总。团队一致认为——"
"一致？"他放下笔，"一致意味着要么方案完美，要么没人敢说真话。"
会议室安静了三秒。

❌ 差的职场对话：
"王总，我们的方案非常好，一定能成功！"
"好的，我批准了，去执行吧。"
```

### 邮件与即时通讯

**职场沟通的潜台词**：
- "收到，我看看" = 优先级不高
- "辛苦了" = 客套，不代表认可
- "这个问题我们线下聊" = 不适合留文字记录
- 回复速度暗示重视程度

## 常见陷阱（Common Pitfalls）

### 1. 金手指过多
- **问题**：主角总有贵人相助，每次危机都有人兜底
- **解决**：贵人可以有，但主角必须自己解决核心问题

### 2. 行业描写悬浮
- **问题**：号称金融小说，但看不到任何金融专业内容
- **解决**：至少深入了解一个行业的核心业务流程

### 3. 反派智商下线
- **问题**：对手犯低级错误，主角轻松获胜
- **解决**：对手要聪明，主角的胜利才有价值

### 4. 升职过快
- **问题**：三年从实习生到VP
- **解决**：参考真实的晋升周期，快速晋升需要合理解释

### 5. 忽视生活面
- **问题**：角色只有工作，没有家庭、感情、个人生活
- **解决**：工作与生活的平衡（或失衡）本身就是好故事

### 6. 说教过多
- **问题**：大段职场鸡汤和人生道理
- **解决**：用故事和角色行为传递观点，而非直接说教

## 爽点设计

### 1. 专业碾压
- **触发时机**：被质疑能力时
- **实现方式**：用专业知识和数据一击制胜
- **示例**：在客户质疑时，当场指出竞对方案的三个致命漏洞

### 2. 逆风翻盘
- **触发时机**：被排挤、被边缘化、项目被砍
- **实现方式**：用被忽视的项目做出超预期成果
- **示例**：被发配到边缘部门，反而做出了公司最赚钱的产品

### 3. 真相揭露
- **触发时机**：长期被冤枉或被暗算
- **实现方式**：关键证据浮出水面，真相大白
- **示例**：被栽赃的项目失败，最终查明是竞争对手泄露了核心数据

## 节奏建议

- **开篇**：用一个职场冲突或关键决策切入，快速展现行业特色
- **中期**：项目推进与人际博弈交织，逐步揭示职场生态的复杂性
- **高潮**：关键项目的成败、晋升的最终角逐、或重大抉择
- **结尾**：不必大团圆，职场的结局往往是一个新的开始

## 总结

职场小说写的是工作，但本质是写人在利益和原则之间的选择。真实的行业细节建立可信度，复杂的人际博弈制造张力，专业能力的展现给人代入感。最好的职场小说让读者既学到东西，又看到人性。
