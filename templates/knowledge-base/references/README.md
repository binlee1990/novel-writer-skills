# 参考资料库使用指南

## 概述

References 参考资料库为特定时代、文化、背景的小说创作提供深度考据资料。

**与其他知识库的区别**：
- **Genres**：类型惯例（如何写言情、悬疑）
- **Craft**：写作技法（如何写对话、场景）
- **References**：背景资料（历史考据、专业知识）← 本库

## 使用方式

### 自动激活（推荐）

提到关键词时自动加载：

```
用户："我要写一个1920年代的故事"
→ 自动加载 china-1920s 参考资料
```

### 手动指定

```
用户："请加载民国参考资料"
用户："我需要1920年代中国的资料"
```

## 使用场景

### 在 /specify 阶段
- 确定故事背景
- 加载对应参考资料
- 了解时代限制和可能性

### 在 /plan 阶段
- 设计符合时代的情节
- 避免时代错误

### 在 /write 阶段
- 查阅具体细节（物价、称呼、礼仪）
- 确保描写真实

### 在 /analyze 阶段
- 检查时代考据错误

## 参考资料格式规范

每个参考资料文件包含：

```markdown
# [主题名称]

## 概览（Overview）
[2-3 段总体介绍]

## 核心信息速查（Quick Facts）
[表格形式的关键数据]

## 详细说明（Detailed Information）
[按子主题组织的详细内容]

## 常见误区（Common Misconceptions）
[历史小说中容易犯的错误]

## 写作应用（Writing Applications）
[如何在小说中使用这些信息]

## 参考来源（Sources）
[资料来源，确保可靠性]
```

## 注意事项

**参考资料是辅助工具，不是限制创造力**
- 历史小说可以合理虚构
- 核心是故事性，而非 100% 准确
- 但避免明显的常识性错误

**注明资料来源**
- 确保信息可靠性
- 方便后续更新和修正
