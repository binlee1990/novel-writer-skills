## Task 2: 设计 show-not-tell.md 内容大纲

**内容结构：**

```markdown
# 展示技巧写作知识库（Show, Don't Tell）

## 快速参考（Quick Reference）
- Show vs Tell 的定义
- 核心原则：让读者体验，而非告知
- 什么时候可以 Tell

## 核心原则（Core Principles）

### 1. 什么是 Show, Don't Tell

#### Tell（讲述）
- 作者直接告诉读者信息
- 抽象、概括性表达
- 读者被动接受

**示例**：
```
她很生气。（Tell）
```

#### Show（展示）
- 通过细节让读者自己推断
- 具体、可感知的描写
- 读者主动参与

**示例**：
```
她一把摔上门，震得墙上的画框晃了三下。（Show）
```

#### 为什么 Show 更好？
1. **代入感强** - 读者"看到"而非"被告知"
2. **情感深刻** - 自己推断的结论更有冲击
3. **画面感足** - 可以想象具体场景
4. **信任读者** - 让读者参与解读

### 2. Show 的五种技巧

#### 技巧 1: 行为展示情绪

**Tell → Show 转换表**：

| 情绪（Tell） | Show 示例 |
|------------|-----------|
| 他很紧张 | 他的手指不停敲击桌面，视线在门口和时钟间来回 |
| 她很悲伤 | 她盯着手机屏幕，泪水模糊了字迹，却一个字也没打出来 |
| 他很骄傲 | 他下巴微抬，目光扫过众人，嘴角勾起一丝弧度 |
| 她很害怕 | 她后退一步,背紧贴着墙,呼吸急促得像刚跑完马拉松 |
| 他很爱她 | 每次她说话,他的目光都会不自觉地落在她身上 |

**原则**：
- ✓ 描写可观察的行为
- ✓ 用肢体语言
- ✓ 展示生理反应
- ✗ 不要说"他感到..."

#### 技巧 2: 对话展示性格

❌ **Tell**：
```
张明是个刻薄的人。
```

✅ **Show**：
```
"你的新发型……挺特别的。"张明盯着她的头顶，
"我是说,不是每个人都敢尝试这种……风格。"
```

**原则**：
- 让角色说话而非作者评价
- 潜台词比直白更有力
- 语气词、停顿、选词都是性格

#### 技巧 3: 感官细节展示环境

❌ **Tell**：
```
这是个贫穷的社区。
```

✅ **Show**：
```
楼道里弥漫着煮白菜的味道,墙皮大片脱落,
露出黑乎乎的水泥。三楼王婆家的门框上,
还贴着去年的红对联,已经褪成了粉红色。
```

**五感检查清单**：
- [ ] 视觉（看到什么）
- [ ] 听觉（听到什么）
- [ ] 嗅觉（闻到什么）
- [ ] 触觉（感觉如何）
- [ ] 味觉（尝到什么）

**原则**：
- 每个场景至少用 2-3 种感官
- 细节必须服务于氛围或情节
- 避免"看上去很……"的表达

#### 技巧 4: 比喻和意象

❌ **Tell**：
```
时间过得很快。
```

✅ **Show**：
```
她低头看手机,再抬头时,窗外已经暗了。
```

**原则**：
- 用具体事物暗示抽象概念
- 比喻要新鲜（避免陈词滥调）
- 意象要符合角色视角

#### 技巧 5: 场景重现

❌ **Tell**：
```
他们童年时经历了很多苦难。
```

✅ **Show**：
```
"还记得那年冬天吗?"她问。
"哪年?"
"就那年啊,你把棉袄给我,自己只穿衬衫,
冻得嘴唇发紫还说不冷。"
他沉默了。
```

**原则**：
- 重现关键场景而非概括
- 用对话或闪回
- 选择最有代表性的细节

### 3. 什么时候可以 Tell

#### Tell 的合理使用场景

1. **过渡信息**
   ```
   接下来的三个月,他每天重复同样的训练。
   ```

2. **次要信息**
   ```
   李明是个会计,在公司工作了五年。
   （如果李明不是主要角色,Tell 即可）
   ```

3. **节奏控制**
   ```
   她累了。
   （简短的 Tell 可以加快节奏）
   ```

4. **在 Show 之后总结**
   ```
   她一把摔上门,转身离开。
   他就这样失去了她。
   ```

**原则**：
- 关键情节必须 Show
- 次要信息可以 Tell
- Show 为主,Tell 为辅

### 4. 常见的 Tell 词汇警报

#### 情绪类 Tell 词
- 感到、觉得、似乎、好像
- 生气、高兴、悲伤、害怕
- 紧张、焦虑、兴奋、失望

#### 性格类 Tell 词
- 是个……的人
- 性格……、脾气……
- 总是、从不、一直

#### 评价类 Tell 词
- 很美、很丑、很好、很坏
- 非常、特别、极其、十分

**替换策略**：
每次用这些词时,问自己：
> "能否用一个可观察的细节代替？"

### 5. Tell → Show 实战训练

#### 练习 1: 情绪转换

将以下 Tell 改为 Show：
1. 他很嫉妒 → ?
2. 她很尴尬 → ?
3. 他很孤独 → ?

**参考答案**：
1. 他紧盯着她和那个男人,手里的酒杯捏得咯吱作响
2. 她耳根发红,视线在地板砖的花纹上游移,不知道手该往哪儿放
3. 餐桌上只摆了一副碗筷,电视开着,声音却关了

#### 练习 2: 场景转换

Tell: 这是个豪华的酒店大堂。

Show: ?

**参考答案**：
```
水晶吊灯垂下来,离地面至少有三层楼高。
大理石地面光可鉴人,她的高跟鞋踩上去,
发出清脆的咔哒声,在空旷的大堂里回荡。
门童穿着烫得笔挺的制服,白手套一尘不染,
为每个进门的客人拉开黄铜把手的玻璃门。
