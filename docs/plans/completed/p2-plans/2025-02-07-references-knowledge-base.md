## 背景与动机

### 问题

**场景 1：历史题材创作困难**
```
作者想写民国背景小说：
- 不清楚 1920 年代的社会结构
- 不知道当时的日常生活细节
- 不了解军阀混战的历史背景
- 担心考据错误影响可信度
```

**场景 2：缺乏真实细节**
```
作者写现代都市小说：
- 职场细节不真实
- 商业逻辑不合理
- 法律程序不准确
```

**场景 3：文化背景不足**
```
作者写异国题材：
- 文化习俗不了解
- 礼仪规范不清楚
- 容易出现文化冲突
```

```markdown
# 1920年代中国概览

## 概览

1920年代的中国正处于军阀混战时期（1916-1928），是从清朝覆灭到国民政府统一之间的过渡阶段。这是一个充满动荡、矛盾和变革的时代。

**核心特征**：
- 政治：军阀割据，中央政府名存实亡
- 经济：半殖民地半封建经济，列强势力强大
- 社会：新旧思想激烈碰撞，五四运动余波
- 文化：新文化运动高潮，传统与现代并存

## 核心信息速查

| 维度 | 信息 |
|------|------|
| **时间跨度** | 1920-1929（北洋军阀后期） |
| **主要政权** | 北洋政府（名义统一）、各地军阀实际控制 |
| **主要城市** | 北京（首都）、上海（经济中心）、广州（革命基地） |
| **货币** | 银元（袁大头）、铜元、各地军用票 |
| **交通** | 铁路、轮船、人力车、马车（汽车极少） |
| **通讯** | 电报、邮政、电话（大城市） |
| **识字率** | 约 20-30%（城乡差距极大） |

## 主要历史事件时间线

**1920年**：
- 直皖战争，直系胜利

**1922年**：
- 第一次直奉战争

**1924年**：
- 第二次直奉战争
- 孙中山改组国民党

**1926年**：
- 国民革命军北伐开始

**1927年**：
- "四一二"政变
- 南京国民政府成立

**1928年**：
- 张作霖被炸死
- 东北易帜，北伐完成

## 社会分层

### 上层阶级（5-10%）
- **军阀及家族**：掌握实权
- **官僚买办**：依附列强的商人
- **大地主**：传统乡绅
- **新兴资本家**：民族工业家

### 中产阶级（15-20%）
- **小地主**：乡村小康
- **知识分子**：教师、律师、医生
- **小商人**：店铺老板
- **技术工人**：高级工匠

### 底层民众（70-80%）
- **农民**：绝大多数人口
- **城市贫民**：工人、小贩、力工
- **流民**：战乱难民

## 城乡对比

| 维度 | 城市（上海/北京） | 乡村 |
|------|------------------|------|
| 建筑 | 西式洋楼、石库门 | 土坯房、青砖瓦房 |
| 服饰 | 旗袍、西装、学生装 | 长衫、粗布衣 |
| 交通 | 电车、人力车、汽车 | 步行、牛车、驴车 |
| 教育 | 新式学堂、教会学校 | 私塾或文盲 |
| 娱乐 | 电影、戏院、舞厅 | 庙会、戏班 |
| 饮食 | 中西餐厅、咖啡馆 | 自给自足、粗茶淡饭 |

## 常见误区

❌ **误区 1：所有地方都很现代化**
- 现实：只有上海、北京等大城市有现代化元素
- 绝大多数地区仍是传统农业社会

❌ **误区 2：女性普遍独立自由**
- 现实：只有大城市少数受教育女性
- 农村女性仍受传统束缚

❌ **误区 3：战乱不断，到处打仗**
- 现实：战争主要在特定地区，大部分地方日常生活仍在继续

❌ **误区 4：人人都说白话文**
- 现实：知识分子推广白话文，但普通百姓仍说方言

## 写作应用

### 设定背景时
- **选择具体年份**（1920-1928 差异很大）
- **选择具体地点**（上海 vs 农村 = 两个世界）
- **考虑主角阶层**（决定生活细节）

### 塑造角色时
- **教育程度**：受教育 = 稀缺资源
- **职业选择**：符合时代可能性
- **思想观念**：新旧交织的矛盾

### 情节设计时
- **利用时代冲突**：传统 vs 现代、中国 vs 西方
- **利用社会问题**：战乱、贫富差距、性别不平等
- **避免时代错误**：技术、制度、观念

## 参考来源

- 《剑桥中华民国史》
- 《北洋军阀史》
- 《民国风云人物》
- 各地方志、回忆录
```

```markdown
# 1920年代中国日常生活

## 衣（Clothing）

### 男性服饰
- **长衫**：知识分子、商人的日常装
- **西装**：高级官员、洋行职员
- **短打**：工人、农民
- **学生装**：中山装的前身

### 女性服饰
- **旗袍**：1920年代开始流行（早期较宽松）
- **袄裙**：传统女装（保守家庭）
- **学生装**：女学生
- **洋装**：极少数时髦女性

### 发型
- **男性**：
  - 1910年代前：辫子
  - 1920年代：分头、平头
  - 革命者：光头（表示决心）
- **女性**：
  - 传统：盘髻
  - 时髦：短发（极大胆）

## 食（Food）

### 日常饮食（普通家庭）
- **主食**：大米（南方）、面食（北方）
- **蔬菜**：时令蔬菜（无反季节菜）
- **肉类**：猪肉（少量）、鱼（江南）
- **频率**：一日三餐（农民两餐）

### 城市餐饮
- **中餐馆**：分档次（高级酒楼到小饭馆）
- **西餐厅**：上海、北京大城市
- **茶馆**：社交场所
- **小吃摊**：街头常见

### 典型一餐（中产家庭）
- 早餐：粥、咸菜、包子
- 午餐：米饭、两菜一汤
- 晚餐：类似午餐

### 货币与物价
| 商品 | 价格（银元） | 说明 |
|------|-------------|------|
| 大米 | 0.05-0.08/斤 | 约 500g |
| 猪肉 | 0.15-0.20/斤 | 较贵 |
| 鸡蛋 | 0.02/个 | |
| 黄包车 | 0.10-0.20/次 | 短途 |
| 电影票 | 0.30-0.50 | 较奢侈 |
| 月薪（工人） | 10-20 银元 | 熟练工 |
| 月薪（教师） | 30-50 银元 | 小学教师 |

## 住（Housing）

### 城市住宅
- **石库门**（上海）：典型民居
- **四合院**（北京）：传统建筑
- **洋房**：富人居所
- **棚户区**：贫民窟

### 农村住宅
- **青砖瓦房**：殷实人家
- **土坯房**：普通农民
- **茅草房**：贫困家庭

### 租房市场（上海）
- 石库门一间：2-5 银元/月
- 整套洋房：20-50 银元/月

## 行（Transportation）

### 城市交通
- **步行**：最常见
- **人力车**（黄包车）：主要交通工具
- **电车**：上海、北京有轨电车
- **汽车**：极少数，富人专属
- **自行车**：稀有，身份象征

### 长途交通
- **火车**：主要干线（京汉、京沪）
- **轮船**：江河湖泊
- **长途汽车**：极少
- **步行/驴车**：农村主要方式

### 旅行时间参考
- 上海 → 北京：火车 30+ 小时
- 上海 → 南京：火车 8 小时
- 城内：步行 1 小时能到大部分地方

## 娱乐（Entertainment）

### 传统娱乐
- **戏曲**：京剧、地方戏
- **茶馆**：听书、聊天
- **庙会**：节日娱乐
- **赌博**：麻将、牌九

### 新兴娱乐
- **电影**：无声电影为主
- **舞厅**：大城市，时髦场所
- **话剧**：新文化运动产物
- **体育**：足球、篮球（学生）

### 阅读
- **报纸**：《申报》等
- **杂志**：《新青年》等
- **小说**：鸳鸯蝴蝶派、新文学
- **识字率低**：阅读是少数人特权

## 通讯与信息

### 通讯方式
- **信件**：主要方式
- **电报**：紧急事务
- **电话**：大城市富人
- **口信**：熟人传话

### 邮政
- **邮局**：城市有正规邮局
- **民信局**：民间邮递
- **邮资**：平信 0.02-0.04 银元

## 医疗卫生

### 医疗资源
- **西医**：大城市医院
- **中医**：主流选择
- **赤脚医生**：农村
- **巫医**：偏远地区

### 常见疾病
- 传染病：天花、霍乱、伤寒
- 营养不良：普遍现象
- 婴儿死亡率：极高

## 常见误区

❌ **误区：到处都有汽车**
- 现实：1920年代汽车极少，只有大城市少数富人

❌ **误区：普通人也能吃西餐**
- 现实：西餐极贵，只有上流社会

❌ **误区：电话很普及**
- 现实：电话极稀有，多数人一辈子没见过

## 写作应用

### 描写日常场景时
- **细节真实性**：符合时代物价和生活水平
- **阶层差异**：不同阶层的生活天差地别
- **地域差异**：上海 vs 农村 = 两个世界

### 常见场景的正确描写

**吃饭场景**：
```
✓ "他在小饭馆点了一碗阳春面，花了五个铜板"
✗ "他点了一份牛排和红酒"（普通人消费不起）
```

**出行场景**：
```
✓ "拉了辆黄包车，讲好价钱两角银子"
✗ "打了辆出租车"（没有出租车）
```

**通讯场景**：
```
✓ "写了封家书，托邮差带回老家"
✗ "拿起电话就打给了远在乡下的父母"（不现实）

```markdown
# 参考资料库使用指南

## 概述

References 参考资料库为特定时代、文化、背景的小说创作提供深度考据资料。

**与其他知识库的区别**：
- **Genres**：类型惯例（如何写言情、悬疑）
- **Craft**：写作技法（如何写对话、场景）
- **Requirements**：特殊要求（反 AI、快节奏）
- **Styles**：写作风格（网文风、文学风）
- **References**：背景资料（历史考据、专业知识）← 本库

## 已有资料

| 参考库 | 适用题材 | 包含内容 | 字数 |
|--------|---------|---------|------|
| china-1920s | 民国小说 | 军阀、社会、日常生活、文化 | 4000+ 行 |
| tang-dynasty | 古代/仙侠 | 官制、诗歌、日常生活 | 3000+ 行 |
| modern-workplace | 都市职场 | 企业、法律、商业 | 2000+ 行 |
| cultivation-world | 修仙网文 | 境界、门派、法宝 | 1500+ 行 |

## 使用方式

### 自动激活（推荐）

提到关键词时自动加载：

```
用户："我要写一个1920年代的故事"
→ 自动加载 china-1920s 参考资料
```

### 手动指定

```
用户："请加载唐朝参考资料"
用户："我需要现代职场的资料"
```

## 使用场景

### 在 /specify 阶段
- 确定故事背景
- 加载对应参考资料
- 了解时代限制和可能性

### 在 /plan 阶段
- 设计符合时代的情节
- 避免时代错误

### 在 /write 阶段
- 查阅具体细节（物价、称呼、礼仪）
- 确保描写真实

### 在 /analyze 阶段
- 检查时代考据错误

## 如何贡献

欢迎贡献新的参考资料：
1. 选择一个时代/文化/领域
2. 按照格式规范编写
3. 提交 PR

## 注意事项

⚠️ **参考资料是辅助工具，不是限制创造力**
- 历史小说可以合理虚构
- 核心是故事性，而非 100% 准确
- 但避免明显的常识性错误

⚠️ **注明资料来源**
- 确保信息可靠性
- 方便后续更新和修正
```
