# 起点中文网格式规范

## 概述

起点中文网（Qidian）是中国最大的网络文学平台之一。本文档定义了从 Markdown 转换为起点格式的规则。

---

## 转换规则

### 1. 章节标题

```
原始 Markdown：# 开端
转换结果：第一章 开端

原始 Markdown：# 转折
转换结果：第二章 转折
```

**规则：**
- `# 标题` -> `第X章 标题`
- 自动按顺序编号（第一章、第二章...）
- 如果原标题已包含编号（如 `# 第一章 开端`），保持原样
- 编号使用中文数字（一、二、三...）

### 2. 正文格式

**段落处理：**
- 每段开头空两个全角空格（`\u3000\u3000`）
- 段落之间不空行
- 去除所有 Markdown 格式标记

**格式标记去除：**

| Markdown | 转换结果 |
|----------|---------|
| `**粗体**` | 粗体 |
| `*斜体*` | 斜体 |
| `> 引用` | 引用内容（去除 > 标记） |
| `` `代码` `` | 代码 |
| `[链接](url)` | 链接文字 |
| `![图片](url)` | （移除） |

### 3. 引号转换

```
原始：他说："你好。"
转换：他说：「你好。」

原始：她想：'这是什么意思？'
转换：她想：『这是什么意思？』
```

**规则：**
- `""` -> `「」`（双引号 -> 直角双引号）
- `''` -> `『』`（单引号 -> 直角单引号）
- `""` -> `「」`（中文双引号 -> 直角双引号）

### 4. 场景分隔

```
原始：---  或  ***  或  ___
转换：（空行）
```

### 5. 特殊处理

- 省略号：`...` -> `……`（中文省略号）
- 破折号：`--` -> `——`（中文破折号）
- 数字：保持原样

### 6. 编码和输出

- 编码：UTF-8（无 BOM）
- 输出：单文件 TXT
- 文件名：`{书名}.txt`
- 换行符：`\r\n`（Windows 风格，兼容性最好）

---

## 输出格式示例

```
第一章 开端

　　李明站在青云宗的山门前，抬头望着那高耸入云的石阶。

　　「你就是新来的弟子？」一个声音从身后传来。

　　他转过身，看到一个身穿白衣的少女正打量着他。

　　「是的，我叫李明。」他拱手行礼。

第二章 入门

　　清晨的阳光洒在练功场上，李明已经站了一个时辰。

...
```

---

## 字数限制

- 单章建议：2000-4000 字
- 单章上限：无硬性限制，但超过 5000 字建议拆分
- 全书无上限
