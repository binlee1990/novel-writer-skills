# 晋江文学城格式规范

## 概述

晋江文学城（JJWXC）是中国知名的女性向网络文学平台。本文档定义了从 Markdown 转换为晋江格式的规则。

---

## 转换规则

### 1. 章节标题

```
原始 Markdown：# 开端
转换结果：<h2>开端</h2>

原始 Markdown：# 第一章 开端
转换结果：<h2>第一章 开端</h2>
```

**规则：**
- `# 标题` -> `<h2>标题</h2>`
- 保留原始标题格式（不强制编号）
- 章节标题居中显示

### 2. 正文格式

**Markdown -> HTML 转换：**

| Markdown | HTML | 说明 |
|----------|------|------|
| 普通段落 | `<p>　　段落内容</p>` | 段首空两格 |
| `**粗体**` | `<b>粗体</b>` | 保留粗体 |
| `*斜体*` | `<i>斜体</i>` | 保留斜体 |
| `---` | `<br/><br/><center>***</center><br/>` | 场景分隔 |
| `> 引用` | `<blockquote>引用</blockquote>` | 引用块 |
| 空行 | `<br/>` | 换行 |

### 3. 引号

- 保持中文引号 `""` `''`
- 不转换为直角引号

### 4. 特殊标记

晋江支持的特殊标记：

| 标记 | 用途 | 示例 |
|------|------|------|
| `<b>` | 粗体 | `<b>重要内容</b>` |
| `<i>` | 斜体 | `<i>内心独白</i>` |
| `<br/>` | 换行 | 段落间分隔 |
| `<center>` | 居中 | `<center>***</center>` |

**不支持的标记（需移除）：**
- `<img>` - 图片
- `<a>` - 链接
- `<script>` - 脚本
- `<style>` - 样式
- 其他复杂 HTML 标签

### 5. 编码和输出

- 编码：UTF-8
- 输出：单文件 HTML
- 文件名：`{书名}.html`

---

## 输出格式示例

```html
<h2>第一章 开端</h2>

<p>　　李明站在青云宗的山门前，抬头望着那高耸入云的石阶。</p>

<p>　　"你就是新来的弟子？"一个声音从身后传来。</p>

<p>　　他转过身，看到一个身穿白衣的少女正打量着他。</p>

<p>　　"是的，我叫李明。"他拱手行礼。</p>

<br/><br/><center>***</center><br/>

<h2>第二章 入门</h2>

<p>　　清晨的阳光洒在练功场上，李明已经站了一个时辰。</p>
```

---

## 字数限制

- 单章建议：2000-5000 字
- 单章上限：无硬性限制
- 全书无上限

## 晋江特殊要求

- 防盗章：晋江有防盗机制，正式发布时需在平台编辑器中操作
- 本导出格式适用于：草稿准备、内容备份、批量上传前的格式化
